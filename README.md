# elena-ads

Лендинг специалиста по интернет-рекламе (Яндекс Директ, Авито).

**Ссылка:** [elena-ads](https://elena-ads.vercel.app)

## Стек

Чистый HTML/CSS/JS — без фреймворков и зависимостей. Один файл `index.html` + `vercel.json`.

## Структура страницы

- **Hero** — заголовок, описание, статистика, 4 анимированных карточки платформ с паттерном курсоров
- **Услуги** — Яндекс Директ (поиск + РСЯ), Авито, аудит и ведение
- **Кейсы** — 3 результата с цифрами на тёмном фоне
- **Как работаю** — 4 шага от брифа до отчёта
- **Обо мне** — фото, опыт, принципы работы
- **Цены** — разовая настройка и ежемесячное ведение, кнопка «Обсудить проект»
- **Бонус** — 5 бесплатных PDF-материалов для скачивания: инструкция по ведению, чек-лист, словарь терминов, разбор ошибок, шаблон таблицы учёта
- **Контакты** — Telegram, телефон, Email, MAX

## Адаптивность

Сайт оптимизирован для всех устройств:

- **Десктоп (>900px)** — двухколоночный hero, анимированные карточки лесенкой с тенями, сетка услуг 2×2, кейсы 3 в ряд
- **Планшет и мобильный (<900px)** — гамбургер-меню, одноколоночный hero, услуги и кейсы в одну колонку
- **Мобильный (<600px)** — все секции в одну колонку, hero только текст без карточек

Шрифты масштабируются через `clamp()` — заголовок hero от 2.8rem до 4.2rem.

## SEO

- `<title>`, `<meta description>`, `<meta keywords>` — расширенные с ключевыми запросами
- `<link rel="canonical">`, `<meta name="robots">`, `<meta name="theme-color">`
- Open Graph (og:title, og:description, og:url, og:locale, og:site_name)
- Twitter Card
- Schema.org JSON-LD — три блока:
  - `Person` — имя, должность, контакты, цены на услуги с `PriceSpecification`
  - `ItemList` — все 5 PDF-материалов с прямыми URL
  - `FAQPage` — 4 вопроса/ответа (цены, сроки, аудит) — могут показываться в выдаче Яндекса и Google
- Семантическая разметка: `<header>`, `<nav>`, `<section>`, `<footer>`
- Иерархия заголовков h1 → h2 → h3
- `alt` у фото, `aria-label` у всех ссылок на PDF

## UX и дизайн

- Шрифты: Playfair Display (заголовки) + Geologica (текст)
- Акцентный цвет: глубокий синий `#2952a3`
- Паттерн курсоров мыши с кругами кликов — на hero-секции и блоке Бонус
- Анимация hero-карточек: парение с анимированной тенью (`float` keyframes), у каждой свой delay
- Гамбургер-меню на планшете и мобильном — анимируется в крестик, закрывается по Escape и клику вне меню
- Модальное окно контактов — открывается по кнопке «Обсудить проект», показывает все способы связи
- Scroll-reveal: все секции появляются при прокрутке через IntersectionObserver
- Hover-эффекты на услугах, контактах, кнопках навигации

## Производительность

- Нет внешних JS-библиотек — только нативный JS (~40 строк)
- Фото встроено как base64 — не требует отдельного HTTP-запроса
- CSS-переменные для всех цветов — легко менять тему
- Минимум HTTP-запросов: HTML + 1 шрифт с Google Fonts

## Как обновить контент

| Что менять | Где искать в коде |
|---|---|
| Имя, должность | `<title>`, `<meta>`, JSON-LD, nav, footer |
| Телефон | `href="tel:..."`, текст карточки контакта, footer |
| Email | `href="mailto:..."`, текст карточки |
| Telegram | `href="https://t.me/..."` (3 места) |
| MAX | `href="https://max.ru/..."` |
| Фото | `src="data:image/jpeg;base64,..."` в блоке `#about` |
| Кейсы | Секция `#cases`, три блока `.case-item` |
| Цены | Секция `#pricing`, `.price-num` и `.price-includes` |
