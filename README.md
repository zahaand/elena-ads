# elena-ads.vercel.app

Лендинг специалиста по интернет-рекламе (Яндекс Директ, Авито).

## Стек

Чистый HTML/CSS/JS — без фреймворков и зависимостей. Один файл `index.html`.

## Структура страницы

- **Hero** — заголовок, описание, статистика, карточки платформ
- **Услуги** — Яндекс Директ (поиск + РСЯ), Авито, аудит и ведение
- **Кейсы** — 3 реальных результата с цифрами
- **Как работаю** — 4 шага от брифа до отчёта
- **Обо мне** — фото, опыт, принципы
- **Цены** — разовая настройка и ежемесячное ведение
- **Контакты** — Telegram, телефон, Email, MAX

## SEO

- `<title>`, `<meta description>`, `<meta keywords>`
- `<link rel="canonical">`
- Open Graph (og:title, og:description, og:url)
- Schema.org JSON-LD (Person + Service)
- Семантическая разметка: `<header>`, `<nav>`, `<section>`, `<footer>`
- Иерархия заголовков h1 → h2 → h3
- `alt` у фото

## Деплой

Хостинг — [Vercel](https://vercel.com). Деплой автоматический при пуше в `main`.
